Selector in HTML

Elements selector

The elements selector selects HTML elements based on the element name

Id Selector

The id selctor use the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element!

Class selector

The class selector select HTML element with a specific class attribute.To select element with a specific class, write a period(.) character, followed by the class name.
